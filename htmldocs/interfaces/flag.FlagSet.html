Interface FlagSet

A FlagSet represents a set of defined flags. The zero value of a FlagSet has no name and has ContinueOnError error handling.

Flag names must be unique within a FlagSet. An attempt to define a flag whose name is already in use will cause a panic.

usage: (() => void)

Type declaration

    • (): void

    • Usage is the function called when an error occurs while parsing flags. The field is a function (not a method) that may be changed to point to a custom error handler. What happens after Usage is called depends on the ErrorHandling setting; for the command line, this defaults to ExitOnError, which exits the program after calling Usage.
